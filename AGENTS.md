# AGENTS

Описание проекта - сайт лендинга сети магазинов детской обуви Cipo и каталог товаров внутри

## Технологии
1. Next.js 16
- максимальное использование серверного рендеринга
- клиентские компоненты на React только для интеративных элементов
- генерация sitemap.xml при сборке
2. каждый компонент и страница (tsx) содержат свой файл CSS и подключают его.
В Global.css содержится базовая стилизация:
- фон везде по умолчанию белый
- базовый цвет - #ef6419
- не используются процессоры CSS, только базовый CSS и nested CSS
- верстка должна быть семантической (например footer называется footer, а не div)

## Страницы и блоки
1. Главная страница и список блоков сверху вниз:
- меню (компонент Header)
- преймущества (компонент AdvantagesList) - в виде карусели картинок на всю ширину и текста
- новинки (компонент NewProductsList) - несколько карточек товаров по горизонтали, плюс ссылку на страницу каталога
- новости и блог (компонент NewsList) - несколько карточек новостей сеткой
- магазины (компонент ShopList) - описания магазина, адрес, фото, режим работы, ссылка на 2GIS
- подвал (компонент Footer) - наименование и реквизиты ИП, юридический адрес, ссылка на оферту, ссылка на whatsapp, ссылка на instagram, ссылка на email

Меню и подвал используются в layout на всех страницах

2. Каталог товаров (страница catalog)
2.1 Слева страницы блок фильтров (компонент Filters) - включаюет кнопки: сезон, размер, тип обуви (сапоги, сандалии), текст, цена (минимальная и максимальная)  
Отдельный компонент Next с useClient, получает значения фильтров через API

2.2 Справа страницы - Список товаров (компонент ProductsList) - содержит сетку карточек товаров и пагинацию

3. Страница товара (catalog/[id]) - открывается при нажатии на карточку
Содержит карусель картинок товара, наименование, артикул, материал верха обуви, материал внутри обуви, материал подошвы, цена

4. Страница новости (news/[id]) - открывается при нажатии на карточку
Содержит изображение новости, заголовок, текст новости, дату публикации

## Общие компоненты Next.js
- Карточка товара вертикальная - отдельный компонент Next, используется на главной странице, в каталоге товаров, которые используется в каталоге и на главной странице 
Содержит сверху карточки - 1 главное изображение товара с квадратным соотношением сторон
Снизу карточки - наименование, список доступных размеров, цену 

- Карточка товара горизонтальная - отдельный компонент Next, используется на главной странице, в каталоге товаров, которые используется в каталоге и на главной странице 
Содержит слева карточки - 1 главное изображение товара с квадратным соотношением сторон
Справа карточки - наименование, список доступных размеров, цену

- Карточка новости горизонтальная - отдельный компонент Next, используется на главной странице. Содержит слева карточки - 1 главное изображение новости.
Справа карточки - краткий текст новости и ссылка на страницу новости

- Меню содержит ссылки на те же разделы что и на главной странице + ссылка на каталог + ссылки на whatsapp и instagram.

## Адаптивная верстка
- при размере экрана меньше 500px меню скрывается и появляется гамбургер-кнопка меню, 
- сетка карточек товаров на главной странице и в каталоге товаров изменяется в зависимости от размера экрана

## Визуальный стиль и дизайн-код

### 1. Общая концепция
* **Стиль:** Современный минимализм с уклоном в "Eco-friendly" и "Natural". Дизайн выглядит "чистым", легким и интуитивно понятным.
* **Атмосфера:** Доверительная, экспертная, но при этом теплая и ориентированная на родителей (Care & Health).
* **Сетка:** Просторная (много "воздуха"), использование карточной системы для товаров и блоков с преимуществами.

### 2. Цветовая палитра
* **Основные цвета:** Нейтральные природные оттенки. Преобладание белого фона, основного акцентного цвета.
* **Шрифты:** Тёмно-серый или антрацитовый вместо абсолютно черного для повышения читаемости и мягкости восприятия.

### 3. Типографика
* **Шрифтовая пара:** Использование современных гротесков (Sans-serif) с хорошим межстрочным интервалом. Заголовки четкие, не перегруженные декоративными элементами.
* **Иерархия:** Выраженная структура заголовков (H1-H3), акцент на ключевых тезисах о здоровье стопы.

### 4. Графика и UI-элементы
* **Фотоконтент:** Живые "лайфстайл" фотографии детей и детальные предметные снимки обуви на нейтральном фоне. Минимум студийного глянца, больше естественности.
* **Кнопки и формы:** Скругленные углы (Border-radius), мягкие тени или их отсутствие (Flat-дизайн). Лаконичные иконки.

### 5. Tone of Voice (визуальный)
* Прямолинейность, честность и фокус на пользе. Дизайн не "кричит" о скидках, а приглашает к изучению философии бренда.

## Получаем из API
- /api/productsFilter - списки значений фильтров
- /api/stores - список магазинов
- /api/news - список новостей
- /api/news/[id] - карточка новости
- /api/productsNews - список новых товаров
- /api/products - список товаров согласно заданным фильтрам
- /api/products/[id] - карточка товара
